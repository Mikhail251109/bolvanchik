ИНСТРУКЦИЯ (Windows)

1. Установи Python 3.10–3.12
   https://www.python.org/downloads/windows/
   (галочка Add Python to PATH обязательна)

2. Распакуй архив console_story_bot.zip

3. Открой папку, Shift + ПКМ → Открыть PowerShell здесь

4. Установи зависимости:
   pip install -r requirements.txt

5. Открой bot.py и вставь:
   - Telegram Bot Token
   - OpenAI API Key

6. Запуск:
   python bot.py

7. Пока консоль открыта — бот работает.
